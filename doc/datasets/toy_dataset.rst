مجموعات البيانات التجريبية
=============================

.. currentmodule:: sklearn.datasets

يأتي scikit-learn مع بعض مجموعات البيانات القياسية الصغيرة التي لا تتطلب تنزيل أي ملف من أي موقع ويب خارجي.

يمكن تحميلها باستخدام الدالات التالية:

.. autosummary::

   load_iris
   load_diabetes
   load_digits
   load_linnerud
   load_wine
   load_breast_cancer
